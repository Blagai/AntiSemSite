FAQ

How does it work?

This extension will colour known Jewish-friendly (green) and Anti-semitic (red) users/pages differently.

Which sites are supported?

This extension supports Twitter, Facebook, Reddit, Tumblr, Medium, YouTube, Wikipedia articles, and search engines.

How does it know how to colour each page?

Manual data gathering, I might integrate machine learning or an AI into it at some point if it's not too complicated.

Does it submit the list of pages that appear on my screen?

No, the check is done offline, on your computer.

Which criteria is used to determine the label?

This extension is focused on Anti-Semitic sentiment of all flavours: Anti-Israel "progressivism", religious, alt-right, etc. Although most of the information is about the first group.

Can I see the list of pages and their labels?

Yes, but I'm not sure what ID the browser will give the extension, so I can't tell you how to do it specifically, but if you don't have many extensions, it shouldn't be too complicated to find the data.

Can I use it on my phone?

Not yet, but I do plan on making it possible using Firefox Nightly. Note however that you won't be able to right-click (and therefore set new labels).

Is there a mechanism in place to prevent malicious/fake reports?

Yes. While your overrides are immediately visible to you, changes are included in the publicly visible dataset only if they pass some trustworthiness criteria (including human validation).

What's the origin of this extension's name?

The original Shinigami Eyes extension, made for trans people.

Why did you create this extension?

Both conservative and progressive communities and people have rampant anti-semitism in them. Be it "progressive" anti-semitism (the "Jews don't belong in the Middle East" people), or the straight-up nazi shit, it's anti-semitism nonetheless.

The purpose of this extension is to make Jewish people feel more confident towards people, groups, and pages they can trust, and to highlight possible interactions with the anti-semitic ones.

Is this extension compatible with the original Shinigami Eyes?

As long as a user isn't marked in both extensions, yes. If a user is marked in both extensions, I don't know which will take priority.
